# Features and Functionality

## Current Features (v1.0.1)

### 1. User Authentication and Onboarding

#### Registration and Login
- Email and password authentication
- Secure password hashing with bcrypt
- JWT token-based session management
- Automatic token refresh
- Remember me functionality

#### Comprehensive Onboarding Flow
- Personal information collection (age, gender, height, weight)
- Goal setting (lose weight, maintain, gain weight)
- Activity level assessment
- Target weight definition
- Dietary preferences and restrictions
- Workout frequency
- Automatic calorie and macro calculation based on user profile

### 2. AI-Powered Food Recognition

#### Image Analysis
- Take photo with camera or select from gallery
- Google Gemini AI integration for food identification
- Automatic nutritional estimation
- Portion size detection
- Confidence score for accuracy
- Support for multiple food items in single image

#### Nutritional Data
- Calories
- Macronutrients (Protein, Carbohydrates, Fats)
- Micronutrients (Fiber, Sugar, Sodium)
- Serving size information
- Amount in grams

### 3. Food Logging and Tracking

#### Manual Food Entry
- Search comprehensive food database
- Add custom foods with nutritional information
- Barcode scanning (planned)
- Quick add from recent foods
- Meal type categorization (Breakfast, Lunch, Dinner, Snack)
- Date and time stamping
- Notes and comments

#### Food Database
- Extensive pre-loaded food database
- User-created custom foods
- Saved favorite foods
- Recently logged foods
- Brand-specific items
- Restaurant menu items (planned)

#### Meal Management
- View daily food logs
- Edit logged entries
- Delete entries
- Copy meals to other days
- Meal templates for quick logging
